# CheapShark
A simple app to find the best deals on PC games.

## API Reference
This app is powered by the [CheapShark API](https://apidocs.cheapshark.com/).

## Tech Stack
- Kotlin
- MVVM
- Clean Architecture
- Modularization
- Dagger Hilt
- GSON
- Retrofit
- Room
- Paging 3
- Kotlin Coroutines
- LiveData
- Flow
- Glide
- ViewBinding
- Material Design 3
- Kotlin DSL
